# Network Automation Projects

This repository contains Python-based automation projects focused on network data analysis, API integration, and observability-style reporting using pandas and requests.

---

## Project 1 – Network Metrics Analyzer

Reads network performance data from a CSV file, identifies problem devices based on latency and error thresholds, summarizes issues by region, and exports a report.

**Key Skills:**
- Python
- pandas (filtering, sorting, groupby)
- Data analysis
- CSV report generation

---

## Project 2 – API Data Analyzer

Pulls JSON data from a public API, converts it into a pandas DataFrame, extracts nested fields, filters records, and exports a report.

**Key Skills:**
- Python
- requests (API calls)
- JSON parsing
- pandas data processing

---

## Project 3 – Network Observability Analyzer

Processes network-style telemetry data, applies configurable thresholds for latency, packet loss, and CPU utilization, identifies unhealthy devices, explains issue causes, and generates timestamped reports.

**Key Skills:**
- Python
- pandas
- Network observability concepts
- Threshold-based monitoring logic
- Data-driven troubleshooting
